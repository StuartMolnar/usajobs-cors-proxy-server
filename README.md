# Clone the repository
```git clone https://github.com/StuartMolnar/usajobs-cors-proxy-server.git```<br>
then<br>
```cd usajobs-cors-proxy-server```

# Make the setup script executable
```chmod +x init.sh```

# Execute the setup script
```./init.sh```

# Run the Python script
```sudo python3 app.py```
